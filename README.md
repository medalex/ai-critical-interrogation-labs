## Technical Requirements

- Python 3.9+
- OpenAI Python SDK
- Hugging Face Transformers

Install dependencies:

pip install -r setup/requirements.txt

## Ethical Use

These labs are designed for educational and research use only.
Students must critically verify all AI outputs.
